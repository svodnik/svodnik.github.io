CSS3 Transitions on Hover

Hover — No Transition


.bug {
  opacity: 1;
}

.bug:hover {
  opacity: 0;
}

Hover — With Transition


.bug {
  opacity: 1;
  transition: all .3s ease-in-out;
}

.bug:hover {
  opacity: 0;
}

CSS3 Transitions on Click

Toggle Class — No Transition



JS:

$('button').on('click', function () {
  $('.bug').toggleClass('faded');
});

CSS:

.bug {
  opacity: 1;
}

.faded {
  opacity: 0;
}

Toggle Class — With Transition



JS:

$('button').on('click', function () {
  $('.bug').toggleClass('faded');
});

CSS:

.bug {
  opacity: 1;
  transition: all .3s ease-in-out;
}

.faded {
  opacity: 0;
}
