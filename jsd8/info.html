Intro to Programming

Q: What is a program?

⇒ A program is a set of instructions that tells a computer how to carry-out a task (e.g. find Sarah Connor)

Q: What is programming?

⇒ Programming is the art of writing those instructions in a way a computer can understand

Q: How do I become a programmer?

⇒ Not by learning a particular language, but by learning how to think like a computer

Pseudo Code

• Pseudo Code:

⇒ The process of thinking through a program, step-by-step, without actually writing a line of code

⇒ An outline of a program that can be converted into code

⇒ Allows the programmer to focus on problem solving, not the precise layout of the code and its syntax

⇒ Don't need to know how to code, to write pseudo code!

Pseudo Code (continued)

• Write pseudo code for a program that calculates how many cups of your favorite drink you will consume from this day forward, and prints a response to a user

• Take into account:

⇒ Your favorite drink
⇒ Your current age
⇒ How old you plan to live
⇒ How many times/day you consume this drink
Pseudo Code (continued)
// get user input
favDrink = get user's favorite drink
userAge = get user's age
userLifeSpan = get how old user plans to live
drinksPerDay = get times/day user consumes drink

// calculate
drinksPerYear = drinksPerDay * 365
yearsLeft = userLifeSpan - userAge
totalDrinks = drinksPerYear * yearsLeft

// notify user
